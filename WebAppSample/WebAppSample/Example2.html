  • 4. Using XMLHTTPRequest to Consume a JSON Web Service

XMLHTTPRequest

    XMLHTTPRequest is an object from a library in Javascript that makes it possible to exchange data between a web site and a server. The fact that it has XML in it's name can be a little misleading. I'm not using it to get XML data at all. I actually just exported a javascript object array to a text file, and I'm having it read in that text file from my github site. So long as the site responds that it's available, it'll pull the data from that page, parse it, and then I have a loop set up to spit out the information from the page. It's a really simple example, but it showcases one possible use for this functionality. It's also used to check progress on uploads, update web pages without needing to reload them, send data to a server in the background, and tons of other things.

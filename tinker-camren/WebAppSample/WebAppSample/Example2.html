XMLHTTPRequest

    XMLHTTPRequest is an object from a library in Javascript that makes it possible to exchange data between a web site and a server. The fact that it has XML in it's name can be a little misleading. I'm not using it to get XML data at all. I actually just exported a javascript object array to a text file, and I'm having it read in that text file from my github site. So long as the site responds that it's available, it'll pull the data from that page, parse it, and then I have a loop set up to spit out the information from the page. It's a really simple example, but it showcases one possible use for this functionality. It's also used to check progress on uploads, update web pages without needing to reload them, send data to a server in the background, and tons of other things.

    Manipulating CSS with JavaScript

    Div

    Manipulating CSS Class properties with Javascript is actually pretty simple. There are a few key things to remember when you're doing so however. In CSS, properties that are more than a single word are connected using a dash, such as 'border-radius'. when you're manipulating that property using javascript, the format is changed to camel case. So to change the border-radius using javascript, it would be referenced as borderRadius instead in the JS code

    Web Storage API

    div

    The Web Storage API is a dictionary of sorts, where you are able to store key/value pairs on the local computer. The Session Storage part of this will only store information until that web page is closed. The Local Storage part of it will store information until that information is cleared out. I'm not 100% sure what all will clear it out, but it sounds like a reboot will clear it out. I'm sure clearing your temporary internet files will also clear that out.

    The key/value pair can contain many different data types. The key is generally a string from what I understand, but the value for that key can be anything. In my example I'm creating a custom object in JS, populating it with values, and then storing that object. You can do the same with arrays, integers, floats, etc.

    • https://developer.mozilla.org/en-US/docs/Web/API/Window/localStorage

DOM Manipulation

The DOM (Document Object Model) is a Javascript model that encompasses the entire document structure of the webpage. One of the most common things that web developers do is manipulate the DOM in some way. The DOM is the actual webpage that's display in the window of the web browser. Each HTML element in the HTML code of the document represents a node in the DOM. You can manipulate each of these nodes in the DOM, just as you would if you were rewriting the HTML code. Using the DOM in JavaScript, you can create new elements, select existing elements and edit what's inside of them, edit their styling, append new info, and many other things.
